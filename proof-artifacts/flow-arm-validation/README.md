# Flow Arm Validation Artifacts

This folder contains the two dedicated-device validation jobs referenced in `docs/FLOW_ARM_PROOF_SUMMARY.md`.

Included for each job:

- `job.json` - completed handoff job packet
- `status.json` - completed handoff status sidecar
- `manifest.json` - finalized `handoff-result.v1` result manifest
- `output.jpeg` - Nano Banana Pro output image generated through the Flow Arm path

- `job-4b-flowarm-validation-002`
