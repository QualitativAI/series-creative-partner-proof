# Scope Notes

This repo is a compact proof package, not the full working vault.

It includes enough to evaluate whether the project is real: the final build-plan corpus, working scripts, real handoff artifacts, result manifests, Flow Arm status evidence, a review lightbox with generated images, the system diagram, and short review briefs.

It does not include machine-specific or unnecessary material:

- live `.env` files, tokens, API keys, or OAuth state
- Chroma database files
- the full holdout image set and per-item answer rows
- the full evaluation archive
- raw personal Creative DNA notes
- local Docker/Hermes profile state
- bulky generated-media folders
- the local Director Cockpit prototype, which was outside hackathon scope

Some local filesystem paths remain in build plans because they are part of the real build record. They are not credentials.
